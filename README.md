## Folder Structure

The workspace contains two folders by default, where:

selenium-framework/

```├── src/ : the folder to maintain sources
│   ├── drivers/ : chrome driver 
│   ├── main/
│   │   ├── java/
│   │   └── resources/
│   └── test/
│       ├── java/
│       │   ├── pages/
│       │   │   ├── java files
│       │   │   └── ... (page classes)
│       │   ├── tests/
│       │   │   ├── java files
│       │   │   └── ... (test classess)
│       │   └── utils/
│       │       └── ... some utility files
│       └── resources/
├── lib/```
